AirBnB Dashboard created using Tableau

**Dashboard:** [Click Here](https://public.tableau.com/app/profile/findlay.anstruther.norton2506/viz/AirBnB_17813501872180/Dashboard1?publish=yes)

**Goal:** To create a dashboard in Tableau, following Alex the Analyst's video on Youtube.

**Description:** The dashboard is created using the "Airbnb Listings 2016" Dataset [Click Here](https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset). The dataset includes information on the properties that have been listed on AirBnB within the Seattle area for the year 2016. The information included within the dataset are the zipcode of each property, the price to hire each property, how many times each property has been hired out to customers and how many bedrooms, bathrooms and other information on each property. The 5 different visuals show the Average Price for the amount of bedrooms in a property, the count of how many bedrooms are included within the dataset, the average price per zipcode shown as a histogram and a mapo of the Seattle area, and a time series of the amount of revenue per month within the year 2016

**Skills:** Data Exploration, Data Visualization.

**Tech:** Tableau Public
